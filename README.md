# Saint Elias Orthodox Church — Website (Proposed Redesign)

A proposed new homepage for **Saint Elias Antiochian Orthodox Church**, Arvada, Colorado.

**Live preview:** see the GitHub Pages link in the repository's *Settings → Pages* (or the link shared with you).

## What this is
A redesigned, mobile-first homepage built around what visitors actually need — service times and address up front, a warm welcome from Fr. George Shawareb, a "what to expect on your first visit" guide, livestream, and simple stewardship. Designed in an *ancient-yet-warm* Orthodox visual language (parchment, oxblood, liturgical gold).

## For review by Fr. George
This is a **draft for approval** — nothing is live on the real website yet. A few items are placeholders to confirm before launch:
- Fr. George's photo and welcome message wording
- Exact weekly service schedule (Orthros / Vespers times)
- The correct parish phone number
- Real parish photography (iconostasis, coffee hour, festival)

## Status
Prototype / pending parish approval. Once approved, it can be deployed to `steliasarvada.com`.
